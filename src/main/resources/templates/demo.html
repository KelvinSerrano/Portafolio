ML para predicción del valor de Bitcoin consumiendo Binance API

En este cuaderno de Google Colab, detallo el proceso completo para la predicción: cargar el dataset, proceso ETL para limpiar y preparar los datos, entrenar el modelo con una cantidad de 24 iteraciones (monitoreado para evitar el overfitting o sobreajuste) y, finalmente, incorporar una ventana de 60 días de historial para lograr una predicción lo más real posible al mercado actual.

Lo que encontrarás en este Notebook:
  • Consumo de API pública financiera (nuestro Dataset en vivo).
  • Proceso ETL integral para limpiar y estructurar series de tiempo.
  • Implementación de arquitectura de Redes Neuronales (LSTM) para predicción.
  • Entrenamiento del modelo monitoreando la función de pérdida (Loss).
  • Visualización de gráficos y resultado de la predicción final en USD.
